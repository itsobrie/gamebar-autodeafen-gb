# Gamebar Autodeafen

Automatically mutes Xbox Game Bar party chat when the player reaches a configurable percentage in Geometry Dash.
Automatically unmutes on pause, restart, death, completion, or exiting the level.

### Features
- Customizable deafen / undeafen percent thresholds  
- Live controls inside the pause menu  
- Full Geode settings integration  
- No performance impact  
- Built from scratch for Game Bar users
