# Gamebar Autodeafen

Automatically mutes Xbox Game Bar party chat when the player reaches a specified percentage.
Automatically unmutes on pause, restart, death, completion, or exiting the level.

### Features
- Customizable deafen / undeafen percent thresholds  
- Controls inside the pause menu  
- Full Geode settings integration
